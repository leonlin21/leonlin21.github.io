HEXO 搭建全流程，教程分享

软件安装及环境检查

NODE.js DOWNLOADS

  • 以默认选项安装即可
  • 检查是否安装成功
    • win+r 打开 cmd
    • 依次输入命令检查，如有显示版本号，则说明安装成功
      • node -v
      • npm -v

Git 百度网盘 提取码：ov1c

  • 以默认选项安装即可
  • 检查是否安装成功
    • win+r 打开 cmd
    • 使用命令 git --version 检查版本号

安装 HEXO

  • win+r 打开 cmd
    • 使用命令 npm install hexo-cli -g 进行安装
    • 如安装速度太慢，或者安装失败，可依次使用下列命令进行安装
      • npm install -g cnpm --registry=https://registry.npm.taobao.org
      • cnpm install hexo-cli -g
  • 检查是否安装成功
    • win+r 打开 cmd
    • 使用命令 hexo -v 检查版本号
    • {:height 337, :width 617}

搭建 Github 仓库

  • 注册 Github
  • 登录 Github
  • 新建一个仓库
  • 使用你的注册名填写信息
    • 注册名就是你 Github 的 ID
    • 格式固定为 注册名.github.io
    • 按照下图 1、2、3 顺序依次操作

新建本地博客文件夹

  • 在电脑本地任意位置新建一个文件夹
  • 这个文件夹是用来存放博客文件的
  • 新建完成后，进入到这个文件夹中

将 Github 仓库与本地 HEXO 绑定

  • 进入本地博客文件夹后
    • 鼠标右键文件夹空白处，然后点击 Git Bash here ，打开 Git 窗口
    • 修改这条命令 ssh-keygen -t rsa -C "邮件地址" ，将 邮件地址 替换成你 GitHub 的邮箱，注意这里不要删掉双引号，然后复制这条命令
    • 打开 Git 窗口，然后右键点击 Paste ，粘贴刚才复制的命令
    • 然后依次敲 4 次回车键，直到出现下图即可
  • 进入 C 盘找到 .ssh 文件夹
    • 路径为： C盘-用户-你的用户名-.ssh文件夹
    • 找到 id_rsa.pub 文件，右键后使用记事本打开
    • 全选复制记事本中的内容
  • 打开 Github
    • 进入设置界面，路径为： 用户 - Settings - SSH and GPG Keys
    • 点击 New SSH keys
    • 输入 Title 名称（此处名称可自定义）
    • 将刚才从记事本复制的内容粘贴到 Key 下方的框框中
    • 然后点击 Add SSH Keys
  • 测试 SSH 是否绑定成功
    • 复制命令 ssh -T git@github.com
    • 打开 Git 窗口，右键 Paste 粘贴命令，然后回车
    • 回车后输入 yes 命令，然后再回车
    • 出现如下图提示，即表示成功

部署本地博客文件

  • 进入本地博客文件夹
  • 右键空白处点击 Git Bash here ，打开 Git 窗口
  • 使用命令 hexo init 进行初始化安装，出现提示 INFO start blogging with Hexo 即表示成功
  • 使用命令 hexo s 生成本地页面
  • 打开网页浏览器，输入地址 localhost:4000 ,如果能成功进入，即表示本地博客搭建成功

发布博客文章

  • 进入本地博客文件夹，找到文件 _config.yml ，右键这个文件，然后使用记事本将它打开
  • 将记事本文件拉到最下面，找到 deploy ,然后按照下列格式进行配置
    • type: git
    • repository: 你的仓库地址
      • 仓库地址需要进入你在 github 新建的博客仓库
      • 找到 Code ，点击后复制地址即可
    • branch: main
    • 配置好之后，保存记事本，然后关闭
    • 注意：这里的所有冒号都是英文格式下的冒号，并且冒号后面要有一个空格
  • 使用命令 Git Bash here 打开 Git 界面
    • 复制命令 npm install hexo-deployer-git --save ，右键 Git 窗口后，Paste 粘贴后回车
    • 然后等待安装完毕
    • 复制命令 hexo g ，Paste 后回车
    • 复制命令 hexo d ，Paste 后回车 ，这时会弹出一个框，要求你登录 Github
      • username： 这里填 Github 的用户名，不是填邮箱
      • Password: 这里填 Github 令牌，不是填 Github 的密码，令牌获取方式如下
        • 进入 Github - settings - Developer settings
        • 点击 Generate new token
        • Note 这里自定义名称
        • Expiration 这里选择 No expiration
        • Select Scopes 这里将所有权限勾选上
        • 最后点击页面最下方的 Generate token 即可
        • 然后会自动刷新页面，复制页面中的令牌即可
        • {:height 66, :width 706}
      • 然后登录
