用 Notion 写 Wordpress 博客

简单尝试了一下用 Notion 写 Wordpress 文章的可行性，插件用法很简单，但 Pro 版本太贵了。

1、创建一个 Notion API

访问 Notion Integrations ，点击 New integration，输入 API 名称，关联工作区，然后点击 Save

然后就会获得一串密钥，保存这串密钥备用

记得点击右下角的 Save 按钮。

2、创建一个数据库用于文章发布

在 Notion 中创建一个页面，点击右上角的三个小点，选择 Connect to，再选择你步骤 1 中创建的 API

3、安装 WP sync for Notion 插件

搜索并安装这个插件，然后激活

在插件设置中，点击左上角的 Add New ，然后在「Internal integration token」中，填入步骤 1 获得的密钥

在下图位置，搜索步骤 2 创建的页面的标题

勾选 Include children’s pages

其他保持默认，然后点击右上角的发布按钮

4、在 Notion 中添加文章

5、点击 sync 按钮

在插件后台，点击 Sync 按钮，就可以将 Notion 中的文章同步到 Wordpress 后台了
